Assignment Overview

In this assignment, you will build a Movie Data Processor App that demonstrates your understanding of:

  • ES6+ features: arrow functions, object/array destructuring, spread & rest operators, template literals
  • Functional programming using array methods: map, filter, reduce, find, sort
  • Modular JavaScript using ES6 modules

The app will process a dataset of movies and allow you to manipulate, filter, and display data in various ways.
